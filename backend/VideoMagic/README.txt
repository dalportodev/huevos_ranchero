HOW TO RUN FFMPEG:

Click this link: https://www.dropbox.com/s/rbbm5sa5py6nqts/VideoMagic.zip?dl=0

After downloading, unzip the folder. Inside, there should be a Visual Studio solution project file, along with other various folders and files. Open that project file with Microsoft's Visual Studio. There is already a test video in the ExampleVid folder, but feel free to test your own video.

More updates to come. Let me know if you find any problems :)
